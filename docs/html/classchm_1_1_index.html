template<class T = NaiveTemplate>
class chm::Index< T >

Hlavní třída indexu HNSW, která využívá všech ostatních tříd.

Parametry šablon
TTřída určující metodu výběru sousedů, typ seznamu navštívených vrcholů a druh přístupu do paměti při výpočtu vzdálenosti.
Viz také
IndexTemplate.
